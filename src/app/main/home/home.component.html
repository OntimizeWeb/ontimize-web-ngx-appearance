What is Ontimize Web Appearance?

Ontimize Web Appearance is web application that shows a form configured with three different appearances supported by Angular Material:

  • Appearance standard: It shows the input box with an underline underneath it.
  • Appearance fill: It displays the form field with a filled background box in addition to the underline.
  • Appearance outline: It shows the form field with a border all the way around, not just an underline.
